# claude-code-website
A website showing amazing things you can do with claude code

Preview live at **[veynu.com](https://veynu.com)**

## What Is This

A single `index.html` file — no frameworks, no build tools. Every line was generated through natural language prompts using [Claude Code](https://claude.ai/claude-code) (Claude Opus).

## Features

- Canvas particle network with mouse attraction
- Custom cursor with lerp tracking
- 3D card tilt + spotlight on hover
- Animated SVG mesh gradient blobs
- Workflow builder, AI engine, and dashboard mockups
- Scroll-triggered directional animations
- Rotating conic-gradient animated borders
- CSS-only marquee logo bar
- Noise grain overlay, button ripples, floating particles
- Fully responsive + `prefers-reduced-motion` support

## How It Was Built

5 prompts, start to finish:

1. **"Implement this visual overhaul plan"** — Full page from a 14-point design spec
2. **"Add cool animations everywhere"** — 15 extra effects (scroll progress, text scramble, ripples, etc.)
3. **"Fix readability, add more colors"** — Bug fixes + emerald/rose/cyan accents
4. **"Use Font Awesome icons, add features"** — Icon swap + "How It Works" section
5. **"Fix headline animation, make visuals real"** — Clean CSS reveal + realistic UI mockup panels

Each prompt built on the last. No full rewrites after the initial generation.

## Run Locally

Open `index.html` in your browser. That's it.
